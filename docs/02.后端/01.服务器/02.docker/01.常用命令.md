1. docker container run [name] : 运行指定image
::: 注
命令具有自动抓取 image 文件的功能。如果发现本地没有指定的 image 文件，就会从仓库自动抓取。
:::

2. docker run --rm : 停止容器后自动删除容器

3. docker ps : 查看运行的进行

4. docker image ls : 查看本地所有镜像

5. docker container stop [name] : 停止名为[]的container
