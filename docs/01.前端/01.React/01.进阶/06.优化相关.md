# React.memo

`React.memo`是一个[HOC](/pages/1aa093/)，他的结构为：

```js
React.memo(组件,比较函数（可选）)
```

它会比较内部组件的`props`（浅比较），如果**props一致**，它会选择重用上次的渲染内容。

我们编写父组件和子组件，可以看到当`foo`值不同时，子组件会进行渲染，当相同时，强制刷新渲染，但依然重用了子组件。
